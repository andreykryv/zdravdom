# Инструкция по миграции с Node.js на PHP для хостинга Timeweb

## ✅ Что сделано

Ваш бэкенд переписан на PHP с полной совместимостью API. Все endpoints работают так же, как в оригинальном Node.js сервере.

## 📁 Структура PHP-бэкенда

```
php-backend/
├── api.php          # Основной файл (роутинг + обработчики)
├── config.php       # Конфигурация (БД, почта, секреты)
├── db.php           # Подключение к MySQL
├── database.sql     # Дамп базы данных с вашими данными
├── .htaccess.example # Пример конфигурации Apache
└── README.md        # Подробная документация
```

## 🚀 Быстрый старт на Timeweb

### 1. Создайте базу данных в панели Timeweb
- Зайдите в панель → Базы данных MySQL → Создать
- Запишите: имя БД, пользователя, пароль

### 2. Импортируйте данные
- Откройте phpMyAdmin в панели Timeweb
- Выберите вашу БД → Импорт → Загрузите `database.sql`

### 3. Настройте config.php
Откройте файл и замените:
```php
define('DB_NAME', 'ваша_база');
define('DB_USER', 'ваш_пользователь');
define('DB_PASS', 'ваш_пароль');
define('ADMIN_PASSWORD', 'новый-пароль-админа');
define('JWT_SECRET', 'случайная-строка-32+символа');
```

### 4. Загрузите файлы на хостинг
- FTP или файловый менеджер Timeweb
- Загрузите папку `php-backend/` в корень сайта

### 5. Настройте .htaccess
Скопируйте `.htaccess.example` в корень сайта как `.htaccess`

### 6. Проверьте работу
Откройте: `https://ваш-сайт.ru/api/health`
Ответ: `{"ok":true}`

## 📊 Сравнение до и после

| Компонент | До (Node.js) | После (PHP) |
|-----------|--------------|-------------|
| Сервер | Express.js | PHP 7.4+ |
| База данных | JSON файлы | MySQL |
| Аутентификация | JWT (jsonwebtoken) | JWT (встроенный) |

| Хостинг | VPS/Docker | Timeweb Shared |

## ⚠️ Важные замечания

1. **Пароль администратора**: по умолчанию `admin123` — обязательно смените!
2. **Email заявки**: функция `mail()` может не работать на всех тарифах — используйте PHPMailer при необходимости
3. **CORS**: для продакшена укажите конкретный домен вместо `*`
4. **Версия PHP**: требуется 7.4 или выше (есть на всех тарифах Timeweb)

## 📞 Если что-то не работает

1. Проверьте логи PHP в панели Timeweb
2. Убедитесь, что база данных импортирована корректно
3. Проверьте права доступа к файлам (644 для файлов, 755 для папок)
4. Включите отображение ошибок в `config.php` для отладки
